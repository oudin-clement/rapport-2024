Réalisation de l'application web

Objectifs du projet

Le but était de créer une application web pour la saisie des heures de travail des employés, facilitant la gestion et la génération de rapports sur les heures travaillées.

Technologies utilisées

  • Back-end : Node.js avec Express.js
  • Base de données : MongoDB
  • Front-end : HTML5, CSS3, JavaScript avec React

Développement et réalisation

Le développement a inclus la création de fonctionnalités pour la saisie, la modification et la visualisation des heures de travail. L'application permet également de générer des rapports détaillés pour les administrateurs.

Résultats obtenus

Une application fonctionnelle a été livrée, répondant aux besoins de gestion des heures de travail. L'application est facile à utiliser et a simplifié le processus de suivi du temps pour les employés et les gestionnaires.
